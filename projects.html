Intractable Problems:The Modified Traveling Salesman Project This was a project for my Algorithms course: CS170. The project specification can be found here. Brief: The project was a modified version of the famous intractable problem: The Traveling Salesman. Given an input graph where each vertex represents a kingdom, and there exixt edge weights as well as conquering costs for each kingdom; our task was to design and develop an algorithm that minimizes the
Total Cost = ∑ Conquering costs + ∑ Traveling costs.
